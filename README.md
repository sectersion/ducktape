# ducktape

**ducktape** is a budget-friendly, fully open alternative to the :contentReference[oaicite:0]{index=0} — the well-known keystroke-injection tool used in cybersecurity and physical pentesting.  
The goal: COST OPTIMIZATION

Instead of hiding behind proprietary hardware, ducktape uses simple off-the-shelf parts and a dead-simple design you can understand, repair, and modify. Cheap to build, easy to hack on, and perfect for learning physical security without dropping triple digits on a dongle.

I built this because I’m diving deeper into physical access testing, and I refuse to pay luxury-brand prices for tools that are essentially microcontrollers pretending to be premium gear.  
Companies like Flipper and Hak5 make cool stuff — sure — but most of it can be recreated for a fraction of the price if you’re willing to solder and tinker.
